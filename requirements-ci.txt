# CI/CD-specific requirements
# =================================
# This file is used in GitHub Actions workflows instead of requirements.txt
# to avoid disk space issues when installing dependencies.
#
# Key differences from requirements.txt:
# - Uses CPU-only PyTorch instead of CUDA-enabled version
# - Saves ~5GB of disk space by avoiding NVIDIA CUDA libraries
# - Functionally equivalent for testing and linting purposes
#
# For local development, use requirements.txt instead.

# First, install PyTorch CPU-only version to prevent CUDA dependencies
--extra-index-url https://download.pytorch.org/whl/cpu
torch>=2.0.0,<3.0.0

# Then install all other requirements
-r requirements.txt
