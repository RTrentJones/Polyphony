# Provenance — Dracula (renamed eval corpus)

## Source work
- **Title:** *Dracula*
- **Author:** Bram Stoker
- **First published:** 1897
- **Copyright status:** **Public domain in the United States** (published before 1929).
- **Text source:** Project Gutenberg edition (eBook #345). Project Gutenberg's own
  license states that once the Project Gutenberg header, license, and all
  references to the Project Gutenberg trademark are removed, *"you are left with a
  text unrestricted by U.S. intellectual property law."* `evals/tools/rename.py`
  strips that wrapper (`strip_gutenberg`) before any text is used. Equivalent
  CC0 editions are available from Standard Ebooks (dedicated to the public domain
  via CC0 1.0) and may be used interchangeably.

## What lives here
- **`excerpts.txt`** — a **renamed derivative** of four chapters (II, V, VIII, X),
  not the whole novel. Generated by `python -m evals.tools.rename --book dracula`.
- **`aliases.json`** — the rename map + the chapters extracted.
- **`ground_truth.json`** — labels derived from the renamed text
  (`python -m evals.tools.build_gt --book dracula`): cast, per-character gold
  voice lines (train/test), canonical plot beats, continuity-injection specs.

## Why the names are changed
Every character and place is renamed (Mina Harker → Nora Vance, Van Helsing →
Verhoeven, Whitby → Marrow Cove, …). This is for **evaluation integrity only**:
feeding a model the *original* text measures how well it recalls a famous novel
from its training data, not how well Polyphony's pipeline extracts characters,
grounds voice, and plans plot from the text actually in front of it. Renaming
defeats that memorization while preserving the (public-domain) prose and the
ground truth. **It is not an attempt to obscure the source** — this file credits
it plainly.

## Legal summary
The underlying work is US public domain; the Project Gutenberg wrapper is stripped
(or a CC0 Standard Ebooks source is used); only a renamed derivative of a few
chapters is committed — never the raw source, never the whole novel. A renamed
derivative of a public-domain work is itself unrestricted. Only this
public-domain-derived text is ever sent to an LLM during evals.
